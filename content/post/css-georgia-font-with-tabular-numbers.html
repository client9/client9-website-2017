This is called by “old style” or “text figures”

CSS

/* serif tabular numbers for macOS, iOS, debian/ubuntu, windows */
@font-face {
  font-family: "tabular-numbers";
  src: local("charter"), local("bitstream charter"), local("cambria");
  unicode-range: U+0030-0039;
}
font-family:
  /* 1 */ tabular-numbers,   /* tabular numbers for Windows, macOS, iOS */
  /* 2 */ georgia,           /* windows, macOS, iOS */
  /* 3 */ bitstream charter, /* debian & ubuntu */
  /* 4 */ serif;             /* everything else */
