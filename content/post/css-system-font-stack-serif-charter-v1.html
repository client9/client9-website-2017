Charter is a great font and it’s available as an upgrade from Georgia on macOS and iOS without using web fonts.

It was originally designed in 1987 for print.  In 1987 that meant 300 dpi laser printers. That's very close to what we call retina or HiDPI for screens today. They are not exactly the same but Charter holds up well on today's screens.

MacOS and iOS now make Charter available as a system font, so it's great choice.  It's also a free font, and so likely installed on many linux and Windows systems as well.  

The next closest system font is Georgia, which happens to be designed by the same person. Georgia is a touch larger and uses proportional numbers.

CSS

```css

font-family:

  /* 1 */ Charter,

  /* 2 */ Georgia,

  /* 3 */ serif;

```
