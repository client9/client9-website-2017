This system stack is upgrade of the traditional serif font Times New Roman. The Mac and Windows alternates are dense and screen optimized.

The CSS

font-family:
  /* 1 */ Athelas,
  /* 2 */ Cambria,
  /* 3 */ serif;

The Fonts

Athelasis macOS and iOS font that is used in iBooks and the Safari Reader feature.

{{< font-sample "font-family: athelas, monospace;" >}}

Cambria

Cambria was a Windows-only font, but Google Docs now seems to support it.

{{< font-sample "font-family: cambria, monospace;" >}}

Caladea

Update 2018-03: Caladea according to the Debian package:

Caladea is metric-compatible with the Cambria font. This font is a serif typeface family based on Huerta Tipográfica’s Cambo.

However, it is not installed by default according to this Debian article. Maybe Google licensed it for ChromeOS, but it’s not on Google Fonts anymore, and the original ChromeOS bug report seems orphaned.  If you use LibreOffice it is “recommended” but not “required”. I can’t find a true primary source. I didn’t add this to the font stack, but if you have more linux experience it might be a worthwhile addition.
