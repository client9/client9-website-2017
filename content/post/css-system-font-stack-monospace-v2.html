The latest monospace system font stack is based on work by GitHub and Bootstrap

Github 2018

font-family:
    "SFMono-Regular",
    Consolas,
    "Liberation Mono",
    Menlo,
    Courier,
    monospace;

Bootstrap v4

font-family:
    SFMono-Regular,
    Menlo,
    Monaco,
    Consolas,
    "Liberation Mono",
    "Courier New",
    monospace;

Monaco has not been a primary font on Mac OS X since 2009, and has never been part of IOS.   Courier New is absolutely not as good as Courier for the screen.  So I’d vote for the GitHub version, however in practice they are functionally identical.
