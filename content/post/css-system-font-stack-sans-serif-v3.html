The latest system front stack is based on GitHub and Bootstrap v4

What’s Changed?

Github.com has now over 6 months of using a system font stack. Bootstrap (the UI framework) with the v4 release has also switched to a native system font stacks.  Given the scale and scope of these projects, their stacks are likely to work well across a diverse set of audiences.

GitHub 2018

Mark Otto who is behind both GitHub and Bootstrap wrote a nice summary of the rollout of GitHub System Fonts 

font-family:
    -apple-system,
    BlinkMacSystemFont,
    "Segoe UI",
    Helvetica,
    Arial,
    Sans-serif,
    "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol";

Bootstrap V4.0.0

Unsurprisingly, Bootstrap’s stack is mostly similar to what Github uses.  This adds Roboto, and swaps Helvetica for Helvetica Neue.  Otherwise the same.

font-family: 
    -apple-system,
    BlinkMacSystemFont,
    "Segoe UI",
    Roboto,
    "Helvetica Neue",
    Arial,
    Sans-serif,
    "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol";
