The large serif font stack uses a number of more-styled serif system fonts that also have a larger size. While this can be used as body text, they might be best at larger sizes for headlines.

All fonts in this stack use old style, proportional numbers making them inappropriate for numeric tables.

CSS

font-family:
  /* 1 */ "Hoefler Text",
  /* 2 */ Constantia,
  /* 3 */ Georgia,
  /* 4 */ serif;
